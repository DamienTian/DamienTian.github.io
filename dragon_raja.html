Job title: SDK Developer

I developed the first version of PC SDK on Unreal Engine 4 for game Dragon Raja, which is the first PC version MMORPG . The SDK provides user related APIs for game development team to use, the APIs include:

  • Login by username & password
  • Login by email
  • Login via Facebook & Twitter
  • Register by username & password
  • Register by email
  • Change password
  • Logout

I worked with the server team (backend) and game development team (frontend) to build the first PC SDK for the company, the SDK has been deployed on Dragon Raja, and the SDK will be applied on other PC games made by Archosaur Games.

In order to make this project, first I studied how does the HTTP protocol work, and I rewrote the most of user realted API of mobile games by native C++ and Unreal C++. I also created the password encryption and decryption API that works with the company's server for PC game; I studied OpenSSL for a couple of days and created the API.

Furthermore, I also create Unreal Engine 4 blueprint plugin of the SDK, the plugin encapsulate all interfaces in the SDK, game development programmer can use Unreal Engine blueprint graphics interface to add the SDK's functionalities in game without rewrite other interfaces to use the SDK or read the SDK's code. For Facebook and Twitter login, I used and modified Unreal Engine Web Browser Widget, and included it in the plugin which achieved in-app third party login.

Feel free to check my work by downloading PC version of Dragon Raja.
