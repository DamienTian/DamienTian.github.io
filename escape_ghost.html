Escape The Ghost

Run away from ghosts, RUN!!! 

Job title: Lead Developer, Animation Maker

This is my 2019 Northeastern University Game Jam project which is done by a student team in 48 hours. The theme of the Game Jam is escape, so the designers came up with a simple idea, escape from some tangible and terrifying objects, which is ghosts. The game contains 3 complete level and a tutorial level. The goal of this game is finding the portal that can escape from the ghosts OR the ghosts will keep hunting you and kill you. The most of art assets (including main character, ghost, background and tiles) are made by the team artist.  

As the lead developer, I used the knowledge I learned from Ben Tristem and Rick Davidson Unity course Complete C# Unity Developer 2D: Learn to Code Making Games on Udemy to develop the game. I used TileMap system in Unity to develop each level. Three levels was created in 48 hours and went through all team members' tests. I also create all characters and items' animations by using the art assets which is made by the artist.

Feel free to download and play it: Escape The Ghost. (Sorry the game only supports Mac OS for now : ( )
