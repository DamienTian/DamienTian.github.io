SDL GameMaker

Tiny 2D platform game engine made by C++, SDL2 Libraries, QT and QT creator 

Job title: Map editor developer, Sprite editor developer

This is my final project for CS 5850 Building Game Engine at Northeastern University. In this project, we developed 3 seperate tools: SDL gamemaker (playable), sprite editor and map editor of the game. I used C++ and SDL2 libraries to developed editors with customized UI on QT creator, user can use the editors to create animation on a raw sprite image and edit game map. Unfortunately, the editors cannot edit the game content in real time because I falied to create another process which runs the game. However, I learned a lot about how to use QT creator to create game tools and also learn knowledge about multiprocessing.

Feel free to check our work on this page: SDL GameMaker.
