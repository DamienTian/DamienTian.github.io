WireFrame Mode

Job title: Developer

In this project, I created a soft renderer with C++ and no external library was used: I wrote everything by my own, include math library, line drawing method and texture mapping method etc. (not include dot drawing)

The soft renderer supports:

  • Wire frame render
  • Color filled render
  • Texture mapping render
  • Object bounding box
  • Face culling
  • Moveable camera
  • Directional light
  • Load complex .OBJ model and texture

In order to make this project, I studied computer graphics and game engine fundamental knowledge, implemented everything that I need for the project and optimized the code for the renderer to have higher FPS. I learned the process of rasterisation, how to optimize graphics related program, and how to better study the modern graphics API;

Feel free to check my work on this page: Soft Renderer.
